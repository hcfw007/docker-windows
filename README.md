## Usage

### Environment variables

* **`USER_PASSWD`** - user `user` password (default: `Passw0rd`)
* **`VNC_PASSWD`** - VNC password, max 8 chars (default: `Passw0rd`)
* **`VNC_GEOMETRY`** - VNC geometry (default: `1600x900`)
* **`VNC_DEPTH`** - VNC depth (default: `16`)

### Ports

* **5900** - VNC (tigervnc)
* **80** - Web (websockify)
